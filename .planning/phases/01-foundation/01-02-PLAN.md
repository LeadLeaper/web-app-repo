<task type="auto">
  <name>Task 2: Establish modern typography system with responsive sizing</name>
  <files>existing/typography.css</files>
  <action>
Create typography.css with professional font system:

**Font families:**
- --font-sans: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif (system font stack for performance and native feel)
- --font-mono: "SF Mono", Monaco, "Cascadia Code", "Roboto Mono", Consolas, monospace (for code/data display)

**Font weights:**
- --font-weight-normal: 400
- --font-weight-medium: 500
- --font-weight-semibold: 600
- --font-weight-bold: 700

**Font sizes (desktop 1920+):**
- --text-xs: 12px / 1.5 line-height
- --text-sm: 14px / 1.5 line-height
- --text-base: 16px / 1.5 line-height
- --text-lg: 18px / 1.5 line-height
- --text-xl: 20px / 1.4 line-height
- --text-2xl: 24px / 1.3 line-height
- --text-3xl: 30px / 1.2 line-height
- --text-4xl: 36px / 1.1 line-height

**Responsive scaling for laptop (1366-1920):**
Use @media query to reduce sizes by ~10% for 1366-1920 viewports:
- --text-base: 14px
- --text-lg: 16px
- Proportionally adjust all other sizes

**Typography utilities:**
- .text-{size} classes: xs, sm, base, lg, xl, 2xl, 3xl, 4xl
- .font-{weight} classes: normal, medium, semibold, bold
- .text-{color} classes: primary, secondary, tertiary (using neutral-600, neutral-500, neutral-400)

**Base styles:**
- body: font-family: var(--font-sans), font-size: var(--text-base), color: var(--color-neutral-700), line-height: 1.5
- h1-h6: Use semibold weight, neutral-800 color, appropriate sizes from scale
- p: margin-bottom: var(--spacing-md)
- a: color: var(--color-primary), hover: var(--color-primary-hover), no underline by default

Set smooth font rendering:
- -webkit-font-smoothing: antialiased
- -moz-osx-font-smoothing: grayscale
  </action>
