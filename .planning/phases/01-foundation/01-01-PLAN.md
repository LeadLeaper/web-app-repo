Create nav-left.js with jQuery-based interaction logic:

**Hover-to-expand behavior:**
- On mouseenter: Add class .nav-expanded to #left-nav (unless already pinned)
- On mouseleave: Remove class .nav-expanded (unless pinned)
- Transition handled by CSS

**Pin/unpin button:**
- On click: Toggle .nav-pinned class on body element
- When pinned: Remove .nav-collapsed, add .nav-pinned
- When unpinned: Remove .nav-pinned, add .nav-collapsed
- Update button icon/state to reflect current state

**State persistence:**
- On page load: Check localStorage.getItem('navPinned')
- If 'true': Apply .nav-pinned class to body
- If 'false' or null: Apply .nav-collapsed class to body
