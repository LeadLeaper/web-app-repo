## Usage

All SVG icons use `currentColor` for easy theming:

```html
<span class="nav-icon" style="color: #08c;">
  <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 24 24" fill="currentColor">
    <!-- icon path -->
  </svg>
</span>
```

## Icon Specifications
- **Format**: SVG (24x24 viewBox)
- **Fill**: currentColor (inherits text color)
- **Size**: Scales to parent container
- **Source**: Material Design Icons

Each navigation section has alternative icon options for visual variety or preference.
